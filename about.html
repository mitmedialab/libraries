Public Library Innovation Exchange (PLIX)

Public libraries and the MIT Media Lab are kindred spirits. Both support an open, collaborative environment where learning can flourish.

This exchange program supports joint projects between public librarians and Media Lab researchers. Together, we hope to design new tools and programs for public libraries around the world.

Learn more about our activities and community.


Collaborators

PLIX is a project of the Media Lab Learning Initiative, in collaboration with the Knight Foundation and MIT Libraries.
